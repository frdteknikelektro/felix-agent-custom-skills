# Command Reference

Use this index for generic `gh` command mechanics before choosing a task-specific operation.

- [Global flags](global-flags.md) covers shared flags such as `--repo`, `--limit`, `--json`, `--jq`, and `--web`.
- [The `--repo` flag](the-repo-flag.md) explains when to pass an explicit `owner/repo` target.
- [Local operations and project workspace](local-operations-and-project-workspace.md) covers clone, branch, edit, push, and PR workflows.
- [API access](api-access.md) covers direct `gh api` calls when a typed `gh` subcommand does not expose the needed endpoint.

For task-specific commands, prefer the operation reference linked from `SKILL.md` over this index.
